# IndexableSkipList
new data structure learnt, implemented in python  

``` python
SkipList
  find(key) -> value
  at(index) -> value
  range(start_index, end_idx) -> [value]
  size() -> int
  set(key, value)
  remove(key)
```

* key must be int
* using same key in set calls will cover the old value
